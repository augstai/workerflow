# Native Mac Reference Notice

The native Workerflow shell borrows architecture patterns and adapts a small amount of UI/state-machine structure from Clicky by Farza:

- menu-bar-only AppKit host
- non-activating `NSPanel` surfaces
- listen-only global push-to-talk event tap
- compact dark SwiftUI permission and voice-control panel
